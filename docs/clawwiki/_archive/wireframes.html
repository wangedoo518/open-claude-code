正文预览

ClaudeWiki 不是另一个 Claude Code Desktop。

它保留桌面工作台的熟悉交互，但把产品中心移向 Raw → Wiki → Schema 三层。 微信只负责把材料投喂进系统；Obsidian 负责阅读与人工编辑；ClaudeWiki 负责标准化、编译、冲突检测、回写与 Token Broker。

对用户来说，ClaudeWiki 的价值不在于“再开一个会话窗口”，而在于把海量碎片材料变成可维护、可搜索、可追问、可版本化的长期认知资产。
